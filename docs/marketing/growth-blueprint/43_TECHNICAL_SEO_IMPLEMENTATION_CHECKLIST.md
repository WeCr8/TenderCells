# Technical SEO Implementation Checklist

## Core Metadata

Every page needs:

- Unique title tag
- Meta description
- H1
- Canonical URL
- Open Graph title
- Open Graph description
- Social preview image
- Structured internal links

## Technical Files

- robots.txt
- sitemap.xml
- llms.txt
- humans.txt
- security.txt later

## Structured Data

Use schema types:

- Organization
- WebSite
- Product
- SoftwareApplication
- TechArticle
- HowTo
- FAQPage
- Course
- EducationalOrganization
- Dataset
- BreadcrumbList

## Performance

- Compress images
- Lazy load non-critical images
- Use alt text
- Keep mobile navigation clean
- Avoid hiding important text inside images only

## Crawlability

- Public pages should return 200 status
- Avoid requiring login for SEO pages
- Public demo landing page should be crawlable
- Developer docs should be crawlable
- GitHub links should be visible

## Indexing

Submit sitemap to:

- Google Search Console
- Bing Webmaster Tools

## Measurement

Track:

- Organic clicks
- Impressions
- Queries
- Indexed pages
- Top landing pages
- GitHub clicks
- Demo clicks
- 4-H / FFA page visits
