# Crawlability Risk Register

## Risk 1: App Demo Hidden Behind Login

If /app/demo requires authentication, search engines may not understand what it does.

Solution:
Create a public demo landing page with screenshots, features, and crawlable text.

## Risk 2: Developer Page Not Indexed

If /developers is not linked from the homepage or sitemap, engineers may not find it.

Solution:
Add it to nav, footer, sitemap, and homepage CTA.

## Risk 3: GitHub Not Discoverable

If GitHub is not linked publicly, open-source contributors may not find it.

Solution:
Add GitHub CTA everywhere.

## Risk 4: Education Angle Not Visible

If 4-H, FFA, STEM, and homeschool pages do not exist, Tender Cells will rank mainly as homesteading tech.

Solution:
Create dedicated education pages.

## Risk 5: Product-Only Positioning

If search only sees smart coop language, the bigger ecosystem story is lost.

Solution:
Create ecosystem pillar pages.

## Risk 6: No llms.txt

AI crawlers may summarize Tender Cells too narrowly.

Solution:
Add /llms.txt.
