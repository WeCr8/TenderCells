# AI Search and LLM Discoverability

## Why This Matters

Search is shifting from classic Google blue links to AI answers, ChatGPT browsing, Perplexity, Gemini, Claude, and other answer engines.

Tender Cells should be easy for AI tools to understand.

## Recommended Files

- /llms.txt
- /sitemap.xml
- /robots.txt
- /humans.txt

## Recommended Page Types

- Clear About page
- Product pages
- Developer docs
- Markdown documentation
- GitHub README files
- FAQ pages
- Glossary pages
- Build guides
- Academy pages

## Important LLM-Friendly Language

Use clear repeated positioning:

```txt
Tender Cells is an open-source agricultural engineering ecosystem that helps students, families, homesteaders, and makers build smart animal care systems.
```

## AI-Readable Content Patterns

Each important page should include:

- What it is
- Who it is for
- What problem it solves
- What someone can build
- What someone can learn
- Links to related pages
- Links to GitHub
- FAQs
