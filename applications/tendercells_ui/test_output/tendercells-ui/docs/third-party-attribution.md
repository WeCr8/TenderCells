# Third-Party Attribution

TenderCells is an open-source homestead automation platform inspired by existing open-source farming and robotics communities. This file records projects we study, reference, or may integrate with so attribution and licensing stay clear.

## FarmBot

FarmBot is a major inspiration for TenderCells. Their open-source CNC farming ecosystem shows how hardware, software, documentation, developer resources, kits, community, and repairability can work together as one practical platform.

Useful FarmBot concepts we are adapting for TenderCells:

- Visual planning and remote operation.
- Open hardware, software, documentation, and developer resources.
- Hardware kits plus a DIY path.
- Developer APIs, message broker concepts, SDKs, scripting, and simulation workflows.
- Self-hosting/offline considerations.
- Education/community positioning.
- Clear DIY support boundaries.

TenderCells expands the pattern toward animal care, homesteading, mobile enclosures, predator monitoring, product/module registries, property simulation, and hardware-in-loop robotics.

### Code Reuse Policy

Do not copy FarmBot code, diagrams, text, media, or CAD files into TenderCells unless:

1. The exact source repository or asset license has been checked.
2. The imported file keeps the required license header or attribution.
3. The import is recorded in this file or an adjacent `NOTICE`/license note.
4. The code is adapted for TenderCells safety requirements, especially animal proximity, doors/latches, water, motors, E-stop, and unattended operation.

Architecture ideas and product patterns may be referenced freely, but actual source code must follow the license of the specific FarmBot repository or asset.
