### Hands-on lessons (do them in order — each links to "What's next")
1. [🐣 Your First Coop Brain](lessons/00-your-first-coop-brain.md) — **start here, ages 7+.** LEGO-style; flash a board, watch it wake up.
2. [🚪 Door + Basic Roaming Roost](CLASSROOM_DOOR_AND_ROAMING_ROOST.md) — servo door + rover from the OS.
3. [🌡️ Sensors → Automated Tasks](CLASSROOM_SENSORS_AND_AUTOMATION.md) — light sensor; auto dawn/dusk door.
4. [🍽️ Feeder + Waterer](CLASSROOM_FEEDER_AND_WATERER.md) — relay feeding/watering + schedules.
5. [🦅 Build Your Own Smart-Animal Device](CLASSROOM_BUILD_YOUR_OWN_DEVICE.md) — invent a species + threat; live alert.
6. [🤖 Gantry + BOMs](CLASSROOM_GANTRY_AND_BOM.md) — XY robot gantry (coop & duck dock) with parts lists.
7. [🧠 AI + CAD with Fusion MCP](CLASSROOM_AI_CAD_FUSION_MCP.md) — design with AI; model in Fusion over MCP.
