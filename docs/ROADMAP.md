# TenderCells Roadmap

## 0.0.1 - Open Source Foundation

- Product and device registry for official, DIY, printed, and custom hardware.
- Product-aware demo animals for testing without hardware.
- ChickenEye camera and egg simulation workflows.
- Property layout, product placement, and 2D/3D viewport direction.
- Local repair loops, Playwright smoke tests, and TenderCells CLI.
- Website story page with real demo screenshots.
- PWA install/testing audit.

## 0.0.2 - Native App Scaffolding

- Add Capacitor config.
- Scaffold Android and iOS native wrappers.
- Generate Android debug APK for local device testing.
- Document SHA-1/SHA-256 fingerprints for Firebase.
- Document iOS local install path through Xcode.
- Keep signing files out of git.

## 0.0.3 - Hardware Bench Testing

- Register the first Chicken TenderCell hardware device.
- Link controller board, doors/latches, motors, water, feed, and sensors to product registry modules.
- Add hardware-in-loop safety states.
- Verify E-stop and low-speed setup behavior.

## 0.1.0 - Community Builder Alpha

- Publish clearer product schemas and module templates.
- Add import/export for custom products and property layouts.
- Add first community hardware proposal workflow.
- Expand animal health record templates across supported species.

## 0.2.0 - Simulation and Robotics

- Improve browser 2D/3D property simulation.
- Add Isaac Sim/ROS integration examples.
- Support terrain, obstacles, routes, corrals, no-go zones, and mobile product behavior.

## Long-Term

- TenderCells OS edge image.
- Offline/self-hosted mode.
- Firmware flashing flow.
- Kit/prebuilt commerce path.
- Community marketplace for printed modules, product profiles, firmware contracts, and build guides.
