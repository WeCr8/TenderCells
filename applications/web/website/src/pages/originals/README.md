## Conversion Notes

When converting to TSX:
1. Replace template literals with JSX
2. Convert onclick handlers to React event handlers
3. Replace class with className
4. Import React components instead of using raw HTML
5. Use React Router's Link component for navigation
6. Convert form handling to React state management
