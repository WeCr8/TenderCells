# Public Discovery Audit
Date: 2026-06-13

## What Public Search Currently Surfaces

Public search surfaced the main Tender Cells homepage with the following discoverable positioning:

- Tender Cells / TenderCells
- Smart Farming Solutions
- Smart farming solutions for modern homesteaders
- AI-powered chicken coops
- Cattle management
- Open-source agricultural technology

## Current Public Positioning Summary

Tender Cells is currently discoverable as a smart farming / homesteading / open-source agriculture technology brand.

This is useful, but it is narrower than the larger strategic opportunity.

## Publicly Discoverable Strengths

- The root domain is visible in search.
- The homepage communicates smart farming.
- The homepage includes homesteader positioning.
- The homepage includes AI-powered chicken coop language.
- The homepage includes cattle management language.
- The homepage includes open-source agriculture technology language.

## Public Discovery Gaps

The following major ecosystem terms did not appear strongly surfaced during the discovery pass:

- 4-H
- FFA
- STEM education
- young engineers
- future engineers
- agricultural robotics education
- homeschool STEM
- science fair projects
- animal health knowledge base
- computer vision for animal care
- ESP32 farming projects
- Fusion 360 coop design
- open-source smart coop hardware
- GitHub Tender Cells
- Tender Cells developers
- Tender Cells Academy

## Interpretation

Search engines appear to understand Tender Cells as a product/technology site more than an education/community/open-source movement.

The next step is to intentionally create discoverable pages that map to each audience and use case.
