# App Demo Public Positioning

## Goal

The app demo should help three groups quickly understand Tender Cells:

1. Families and homesteaders
2. Students and educators
3. Engineers and contributors

## Suggested URL

```txt
https://tendercells.com/app/demo
```

## Demo Page Purpose

Show the future Tender Cells experience:

- Animal dashboards
- Coop conditions
- Feed status
- Water status
- predator alerts
- camera events
- egg tracking
- health trends
- student project mode
- developer mode

## Recommended Demo Landing Copy

```txt
Explore the Tender Cells Smart Farm Dashboard.

See how students, families, homesteaders, and engineers can monitor animal care, learn real-world engineering, and build open-source agricultural technology together.
```

## Demo Modes

### Family Mode
Simple monitoring and alerts.

### Student Mode
Educational explanations and project milestones.

### Engineer Mode
Sensor data, logs, API, firmware, and hardware status.

### 4-H / FFA Mode
Project tracking, animal records, build logs, and presentation outputs.
