# Tender Cells Community Growth Plan

## Goal
Build the largest open-source homestead automation community.

## Channels
- GitHub
- YouTube
- Reddit
- Facebook Homesteading Groups
- FFA
- 4H
- Maker Communities
- Discord

## 90 Day Plan
### Month 1
- Launch GitHub
- Publish roadmap
- Publish hardware standards
- Publish first build guide

### Month 2
- Weekly build videos
- Contributor onboarding
- Open issues for community

### Month 3
- First beta builders program
- Collect deployment photos
- Publish case studies

## Success Metrics
- GitHub Stars
- Discord Members
- Active Contributors
- Devices Deployed
- YouTube Subscribers
