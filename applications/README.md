# Tender Cells Applications

This directory contains all Tender Cells smart farming applications across multiple platforms.

## Platform Structure

### Web Applications
- Built with modern web technologies (React/Vue/Angular)
- Firebase integration for real-time data
- Google AI and Genkit for intelligent insights
- Progressive Web App (PWA) capabilities

### iOS Applications
- Native iOS development with Swift/SwiftUI
- Firebase SDK integration
- Apple-specific features and design guidelines
- App Store distribution

### Android Applications
- Native Android development with Kotlin/Java
- Firebase SDK integration
- Material Design principles
- Google Play Store distribution

## Technology Stack

### Shared Technologies
- **Backend**: Firebase (Firestore, Authentication, Cloud Functions)
- **AI/ML**: Google AI and Genkit for intelligent insights
- **Real-time**: Firebase Realtime Database
- **Analytics**: Firebase Analytics
- **Notifications**: Firebase Cloud Messaging

### Web Stack
- **Framework**: React with TypeScript
- **Styling**: Tailwind CSS
- **Build Tool**: Vite
- **PWA**: Workbox for offline capabilities

### iOS Stack
- **Language**: Swift
- **UI Framework**: SwiftUI
- **Architecture**: MVVM
- **Dependencies**: CocoaPods/Swift Package Manager

### Android Stack
- **Language**: Kotlin
- **UI Framework**: Jetpack Compose
- **Architecture**: MVVM with LiveData
- **Dependencies**: Gradle

## Development Guidelines

### Code Organization
- Each application maintains its own directory structure
- Shared utilities and components in common directories
- Platform-specific implementations in respective folders

### Firebase Integration
- Shared Firebase project configuration
- Environment-specific configurations (dev, staging, prod)
- Consistent data models across platforms

## Getting Started

### Prerequisites
- Node.js 18+ for web development
- Xcode 14+ for iOS development
- Android Studio for Android development
- Firebase CLI for backend services

### Setup Instructions
1. Clone the repository
2. Install dependencies for your target platform
3. Configure Firebase credentials
4. Follow platform-specific setup guides
