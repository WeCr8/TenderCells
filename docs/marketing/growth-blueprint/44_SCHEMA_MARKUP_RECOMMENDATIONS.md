# Schema Markup Recommendations

## Homepage

Use:

- Organization
- WebSite

## Product Pages

Use:

- Product
- FAQPage
- BreadcrumbList

## Academy Pages

Use:

- Course
- EducationalOrganization
- FAQPage

## Developer Pages

Use:

- TechArticle
- SoftwareSourceCode
- HowTo

## Animal Health Pages

Use:

- Article
- FAQPage
- HowTo where appropriate

## Science Fair Pages

Use:

- HowTo
- EducationalOccupationalProgram
- FAQPage

## GitHub / Open Source Pages

Use:

- SoftwareSourceCode
- TechArticle
- Organization

## Example Product Schema Concepts

Chicken Tender should include:

- Name
- Description
- Brand
- Category
- Audience
- Related products
- Educational use
- Open-source links

## Example Academy Schema Concepts

Tender Cells Academy should include:

- Course name
- Learning outcomes
- Age level
- Skills taught
- Project requirements
